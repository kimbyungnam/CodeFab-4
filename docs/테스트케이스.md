# 1. 정상동작 테스트 (기초 기능 테스트)

**1) 표현식 / 연산자 / 우선순위 / 진리값**
```
// --- 산술, 연산자 우선순위 ---
출력 1 + 2 * 3;        // expect: 7
출력 (1 + 2) * 3;      // expect: 9
출력 10 - 4 - 3;       // expect: 3
출력 8 / 2 / 2;        // expect: 2
출력 -3 + 2;           // expect: -1

// --- 비교, 동등성 ---
출력 1 < 2;            // expect: 참
출력 3 > 5;            // expect: 거짓
출력 10 == 10;         // expect: 참
출력 10 != 5;          // expect: 참

// --- 문자열 연결 ---
출력 "안녕, " + "말랑!";     // expect: 안녕, 말랑!

// --- 숫자 출력 포맷 ---
출력 5;                // expect: 5
출력 5.0;              // expect: 5
출력 3.14;             // expect: 3.14

// --- boolean 리터럴 출력 ---
출력 참;               // expect: 참
출력 거짓;             // expect: 거짓
```

**2) 변수, 할당, 블록 스코프, 변수 shawdowing**

```
// --- 선언 & 사용 ---
변수 a = 10;
변수 b = 20;
출력 a + b;            // expect: 30

// --- 재할당 ---
a = a + 5;
출력 a;                // expect: 15

// --- 블록 스코프 & 변수 shadowing ---
변수 x = "전역";
{
  변수 x = "안쪽";
  출력 x;              // expect: 안쪽
}
출력 x;                // expect: 전역

// --- 안쪽 블록에서 바깥 변수 수정 ---
변수 count = 0;
{
  count = count + 1;
}
출력 count;            // expect: 1

// --- 중첩 스코프 해석 ---
변수 outer = "A";
{
  변수 inner = "B";
  {
    출력 outer + inner;  // expect: AB
  }
}
```

**3) 제어 흐름 (if/else, for)**
```
// --- 만약 / 아니면 ---
만약 (참) 출력 "bbq";       // expect: bbq

만약 (거짓) 출력 "no"; 아니면 출력 "kfc";  // expect: kfc

// --- 아니면은 가장 가까운 만약에 결합 ---
만약 (참)
{
  만약 (거짓) 출력 "kfc";
  아니면 출력 "bbq";        // expect: bbq
}

// --- 반복문 ---
반복 (변수 j = 0; j < 3; j = j + 1) {
  출력 j;
}
// expect:
// 0
// 1
// 2
```

# 2. 에러 검출 테스트 (각 스크립트는 "에러를 내야" 정상)
오류 메시지는 팀에서 정한 메시지를 출력.


**1) 구문 에러**
```
// --- 세미콜론 누락 ---
// 기대: "값 뒤에 ';'가 필요합니다." 류의 메시지
출력 1 + 2
```
```
// --- 닫는 괄호 누락 ---
// 기대: "식 뒤에 ')'가 필요합니다." 류의 메시지
출력 (1 + 2;
```
```
// --- 잘못된 할당 대상 ---
// 기대: "잘못된 할당 대상입니다." 류의 메시지
변수 a = 1;
변수 b = 2;
a + b = 3;
```
```
// --- 표현식이 와야 할 자리에 잘못된 토큰 ---
// 기대: "식을 기대했습니다." 류의 메시지
출력 * 5;
```

**2) Checker Unit 에서 검출하는 에러 (정적에러)**
```
// --- 지역 변수를 자기 초기화식에서 읽기 ---
// 기대: "지역 변수는 자기 초기화식에서 읽을 수 없습니다." 류의 메시지
{
  변수 a = a;
}
```
```
// --- 같은 로컬 스코프 중복 선언 ---
// 기대: "이미 같은 스코프에 같은 이름의 변수가 있습니다." 류의 메시지
{
  변수 a = "hi";
  변수 a = 3;
}
```

**3) 실행중 발생하는 런타임 에러**
```
// --- 정의되지 않은 변수 참조 ---
// 기대: "정의되지 않은 변수 'notDefined'입니다." 류의 메시지
출력 notDefined;
```
```
// --- + 연산자에 숫자/문자열 혼용 ---
// 기대: "피연산자는 둘 다 숫자이거나 둘 다 문자열이어야 합니다." 류의 메시지
출력 1 + "HI";
```
```
// --- 숫자가 아닌 값에 단항 마이너스 ---
// 기대: "피연산자는 숫자여야 합니다." 류의 메시지
출력 -"말랑";
```
