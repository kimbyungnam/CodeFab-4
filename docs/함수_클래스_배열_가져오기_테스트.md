## 키워드 매핑

| 기존 키워드   | 한글 키워드 |
|----------|--------|
| `Func`   | `함수`   |
| `return` | `반환`   |
| `Class`  | `클래스`  |
| `var`    | `변수`   |
| `print`  | `출력`   |
| `This`   | `나`    |
| `init`   | `생성자`  |
| `Super`  | `부모`   |
| `Array`  | `배열`   |
| `Import` | `가져오기` |
| `alias`  | `별칭`   |
| `instanceof` | `타입확인` |

## 1. 함수 선언, 반환, 호출

```txt
함수 add(a, b) {
    반환 a + b;
}

변수 ret = add(3, 7);
출력 ret;
```

## 2. 클래스 선언과 인스턴스 생성

```txt
클래스 Robot {
}

변수 robot = Robot();
```

## 3. 인스턴스에 필드 동적 저장 / 읽기

```txt
클래스 Robot {
}

변수 r = Robot();

r.name = "SpeedRobot";
r.speed = 10;

출력 r.name;      // SpeedRobot
```

## 4. 클래스 내부 메서드와 `나`

```txt
클래스 Robot {
    move(dist) {
        나.position = 나.position + dist;
    }

    report() {
        출력 나.position;
    }
}

변수 r = Robot();
r.position = 0;
r.move(5);
r.report();
```

## 5. 생성자

```txt
클래스 Robot {
    생성자(name, speed) {
        나.name = name;
        나.speed = speed;
    }
}

변수 r = Robot("AndOr", 10);
출력 r.name;
```

## 6. 상속과 부모 메서드 호출

```txt
클래스 Robot {
    move(dist) {
        출력 "move";
    }
}

클래스 SpeedRobot : Robot {
    move(dist) {
        부모.move(dist);
        출력 "Speeeed!";
    }
}

SpeedRobot().move(3);
```

## 7. 인스턴스 타입 확인

`instanceof`(다른 언어의 타입 확인 연산자)와 같이 인스턴스가 특정 클래스인지 확인하는
연산자를 `타입확인`이라는 키워드로 추가할 수 있습니다.

부모 클래스가 있는 경우에도 `참`이 나오도록 합니다.

```txt
클래스 Robot {
    생성자(name) {
        나.name = name;
    }
}

클래스 SpeedRobot : Robot {
    생성자(name) {
        부모.생성자(name);
    }
}

변수 w = SpeedRobot("Sam");

출력 (w 타입확인 SpeedRobot);    // expect: 참
출력 (w 타입확인 Robot);         // expect: 참
```

## 8. 고정 크기 배열

고정 크기 배열을 지원합니다.

배열 생성 시 크기가 확정되며, 인덱스를 통한 읽기와 쓰기를 지원합니다.

```txt
변수 arr = 배열(3);          // 크기 3의 배열 생성 [없음, 없음, 없음]

arr[0] = 10;
arr[1] = 20;
arr[2] = 30;

출력 arr[0];                // expect: 10

변수 i = 2;
arr[i - 1] = 7;

출력 arr[1];                // expect: 7
```

## 9. 배열 런타임 오류

실행 중 배열을 잘못 사용하는 경우 런타임 오류를 발생시킵니다.

대표적으로 다음 경우를 오류로 처리합니다.

- 범위를 벗어난 인덱스 접근
- 인덱스가 숫자가 아닌 경우
- 배열이 아닌 대상에 `[]` 사용
- 배열 생성 시 크기로 숫자가 아닌 값 전달

각 코드는 에러가 발생해야 정상입니다.

```txt
변수 arr = 배열(3);          // [없음, 없음, 없음]

출력 arr[5];                // error: 인덱스 범위를 벗어났습니다.
출력 arr["hello"];          // error: 인덱스는 숫자여야 합니다.

변수 x = 10;
출력 x[0];                  // error: 인덱스 접근은 배열에서만 사용할 수 있습니다.

변수 brr = 배열("hi");      // error: 배열의 크기는 숫자여야 합니다.
```

## 10. 라이브러리 가져오기

외부 파일을 라이브러리처럼 가져오는 기능을 지원합니다.

```txt
가져오기 "파일주소" 별칭 별칭명;
```

예를 들어 `sum.txt` 파일에 다음 코드가 있다고 가정합니다.

```txt
함수 add(a, b) {
    반환 a + b;
}
```

REPL 또는 다른 파일에서는 다음처럼 사용할 수 있습니다.

```txt
가져오기 "sum.txt" 별칭 sum;

변수 a = sum.add(1, 2);
출력 a;                    // expect: 3
```

`sum`은 가져온 파일을 참조하는 별칭이며, `sum.add`처럼 해당 파일에 정의된 함수에 접근할 수 있습니다.
